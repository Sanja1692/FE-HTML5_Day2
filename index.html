HTML

1991

HTML is the HyperText Markup Language used to describe web pages.

HTML 2.0

1995

HyperText is what you use daily to move around on the web — by clicking on certain elements called hyperlinks which direct your browser to the next page.

HTML 3.2

1997

The word “hyper” just means it does not function linearly — meaning you can go to any place on the internet from any other place — which is what allows the modern internet as we know it to function.

HTML 4.01

1999

HTML is not a programming language; it is a markup language.

XHTML

2000

Markup is used to tell your browser how to structure the web pages you visit. A web page consists of a series of elements, which you use to enclose or mark up different parts of the content (like page title, or paragraphs) to make it appear a certain way, or act a certain way.

HTML5

2014

HTML consists of a series of short codes typed into a text-file by the site author — these are the elements.
